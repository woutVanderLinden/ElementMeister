Wi-Fi room commands in Kid A:

NOTE: These commands can only be used in the Wi-Fi room or in PM.

Cloner list:

addcloner (username, fc, ign, notes): Adds a cloner to the cloner list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Owner or cloner whitelist.

removecloner (username): Removes a cloner from the cloner list. Requires Room Moderator.

updatecloner (username, [key: value, key: value]): Updates the information of a cloner. Possible values for keys are: fc, ign, notes. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Cloners can update their own info, and Room Owners or cloner whitelisted users can edit any cloner's info.

purgecloners: Removes all cloners from the list that haven't done a giveaway in the past 30 days. Requires Room Owner.

whitelistcloner (username): Adds the given user to the cloner whitelist, allowing them to use cloner commands. Requires Room Owner.

notifycloners (message): Sets a message that will be sent to all cloners when they log on. Requires Room Owner.

cloned (user): Used to confirm your cloning and update the score on the cloners list.

clonerlog: Shows the log of all confirmed clonings of the past month. Requires Room Driver or cloner whitelist.

clonernote (user, note): Adds a new modnote for user. Requires Room Driver or cloner whitelist.

clonernotes: Gives a link to a page to view all cloner modnotes. Requires Room Driver or cloner whitelist.

Scammer list:

addscammer (main, alts, ign, fc, reason): Adds a scammer to the scammer list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Driver.

removescammer (username): Removes a scammer from the scammer list. Requires Room Moderator.

updatescammer (username, [key: value, key: value]): Updates the information of a scammer. Possible values for keys are: alts, fc, ign, reason, addedby. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Room Drivers can edit any scammer's info.

addscammeralt (username, alt1[, alt2, alt3...]): Adds an alt to the scammer's entry. Requires Room Driver.

checkfc (fc): Checks if the given FC is on the scammer list, on the cloner or trainer list, or if the FC belongs to a user in the room.

Hackmon list:

The identifier for editing and deleting hacked mons is species-day-month-year. Use this in .removehackmon and updatehackmon. It is not needed for adding mons.

addhackmon (species, ot, tid, details, reasoning): Adds a hackmon to the hacked mons list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Driver.

removehackmon (identifier): Removes a hackmon from the list. Requires Room Driver.

updatehackmon (identifier, [key: value, key: value]): Updates the information of a hacked mon. Possible values for keys are: species, ot, tid, details, reasoning, notes. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Driver.

Friend Code list:

addfc (username, friendcode): Adds a Friend Code for a user. Requires Room Driver.

deletefc (username): Removes the Friend Code of a user. Requires Room Driver.

fc (username): Returns the friend code of the entered user.

TSV list:

addtsv (username, tsv): Adds a TSV for a user. Requires Room Driver.

deletetsv (username): Removes the TSVs of a user. Requires Room Driver.

tsv (esv1[, esv2, esv3, esv4]): Checks for matches in the TSV list for entered ESVs.
